# cifra de césar em C
algorítimo da cifra de césar. Peguei o algoritmo base do site https://capivararex.wordpress.com/2016/01/30/cifra-de-cesar-em-linguagem-c-2/

A função  *desCriptografa* gera 25 strings,a partir da string passada como argumento, com a chave indo de de 1 a 25, procure alguma que faça sentido. Quando achar, você saberá a chave da criptografia

A função *criptografa* gera 25 strings da frase criptografada com a chave indo de 1 a 25.
